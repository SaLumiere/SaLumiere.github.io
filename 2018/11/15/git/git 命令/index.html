git 命令

  1. git clone 下载代码
  2. git status 查看状态
  3. git add 加入staged
  4. git commit -m 提交
  5. git log 操作日志
  6. cat 文件名 打开文件
  7. v 文件名 编辑文件
  8. git reset 版本号 回退版本
  9. git tag -a v0.1 -m 新增版本
  10. git checkout 选择分支
  11. git remote 查看地址
  12. git push 提交代码到仓库
  13. git remote –help 查看帮助

git config –global user.name “你的用户名”
git config –global user.email “你的邮箱”

方法2：修改.gitconfig文件

路径: C:\Users\你的用户名\.gitconfig

生成ssh key

输入: ssh-keygen -t rsa -C “你的邮箱”

方法1: 输入 clip < ~/.ssh/id_rsa.pub 会自动

复制ssh key，可以直接粘贴

方法2:在c/Users/Administrator/.ssh/id_rsa)文件找到直接复制
