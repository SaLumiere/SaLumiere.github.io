DB2使用

  • 数据库创建

    1. 打开管理窗口，输入db2start,启动db2数据库

    2.db2 create database 数据库名 创建数据库

  • 数据库连接

    1. db2 connect to 数据库名

  • 数据库删除

    1. db2 drop database SLPORTAL

  • 错误处理

    1. DB2 SQL Error: SQLCODE=-204, SQLSTATE=42704” when connecting to new Controller DB
      解决办法：http://www-01.ibm.com/support/docview.wss?uid=swg21613531

    2. 创建表时出现下列错误，

      [Error Code: -286, SQL State: 42727] DB2 SQL Error: SQLCODE=-286, SQLSTATE=42727, SQLERRMC=8192;DB2ADMIN, DRIVER=4.13.127

      解决办法：https://www-01.ibm.com/support/docview.wss?uid=swg21414318
